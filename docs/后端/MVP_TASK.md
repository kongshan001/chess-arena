# 后端岗 - MVP 阶段工作明细

## 角色配置
- 游戏后端 (资深) × 2人
- 架构/运维 (资深) × 1人

## 技术选型
- **语言**: Python (asyncio)
- **框架**: FastAPI
- **存储**: Redis (缓存) + MySQL (持久化)
- **网关**: Nginx + WebSocket

## 工作内容

- [ ] 设计微服务架构
- [ ] 设计 WebSocket 房间管理
- [ ] 设计战斗同步策略 (帧同步)
- [ ] 搭建开发/测试环境
- [ ] 实现基础网关

### 第2周: 核心战斗服务
- [ ] 实现房间服务 (创建/加入/退出)
