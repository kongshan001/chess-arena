# PM 岗 - MVP 阶段工作明细

## 角色配置
- 项目经理 (资深) × 1人

## 工作内容

### 第1周: 项目启动
- [ ] 组建团队，明确分工
- [ ] 制定详细项目计划
- [ ] 搭建协作工具 (Jira/飞书/文档)
- [ ] 确立沟通机制 (日会/周会)
- [ ] 识别项目风险

### 第2周: 执行推进
- [ ] 每日站会跟进
- [ ] 协调资源解决问题
- [ ] 进度跟踪与预警
- [ ] 需求变更管理

### 第3周: 迭代优化
- [ ] 协调测试与开发
- [ ] 组织评审会议
- [ ] 处理突发问题
- [ ] 阶段复盘

### 第4周: 验收交付
- [ ] 组织验收评审
- [ ] 整理项目文档
- [ ] 团队复盘会议
- [ ] 经验沉淀

|--------|--------|
| 项目计划 | PM |
| 进度周报 | PM |
| 风险跟踪表 | PM |
