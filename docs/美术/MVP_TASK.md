# 美术岗 - MVP 阶段工作明细

## 角色配置
- 角色美术 (资深) × 1人
- 场景美术 (资深) × 1人  
- UI美术 (资深) × 1人

## 工作内容

### 第1周: 风格定义与基础设计
- [ ] 确定整体美术风格 (参考: 二次元+电竞)
- [ ] 设计 UI 配色方案
- [ ] 设计棋子基本造型 (3-5个示范)
- [ ] 设计棋盘视觉风格
- [ ] 输出 UI 组件规范

### 第2周: 核心资源制作
- [ ] 制作 MVP 棋子立绘 (5个基础棋子)
- [ ] 制作棋盘底板 UI
- [ ] 制作操作按钮/图标
- [ ] 制作战斗场景背景 (1个)
- [ ] 制作基础特效 (攻击/受击/死亡)

### 第3周: 资源迭代与优化
- [ ] 根据测试反馈优化立绘
- [ ] 补充棋子待机动画
- [ ] 优化战斗特效表现
- [ ] 补充 UI 动效设计
- [ ] 制作加载界面

### 第4周: 验收支持
- [ ] 配合前端调整资源
- [ ] 产出高优先级资源清单 (后续迭代用)
- [ ] 整理美术资源规范文档
