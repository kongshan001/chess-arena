# QA 岗 - MVP 阶段工作明细

## 角色配置
- 功能测试 (资深) × 1人
- 性能/兼容性 (资深) × 1人

## 工作内容

### 第1周: 测试准备
- [ ] 编写测试用例 (核心功能)
- [ ] 设计测试数据
- [ ] 搭建测试环境
- [ ] 准备测试工具

### 第2周: 功能测试
- [ ] 执行功能测试
- [ ] 提交 Bug 跟踪
- [ ] 回归测试
- [ ] 验收标准检查

### 第3周: 体验测试 + 性能
- [ ] 执行体验测试 (手感/节奏/反馈)
- [ ] 性能测试 (帧率/延迟/内存)
- [ ] 兼容性测试 (主流浏览器/机型)
- [ ] 网络稳定性测试

### 第4周: 验收
- [ ] 完整回归测试
- [ ] 产出测试报告
- [ ] 验收报告

## 交付物清单

| 交付物 | 负责人 |
|--------|--------|
| 测试用例 | 功能测试 |
| Bug 跟踪表 | 功能测试 |
| 性能测试报告 | 性能测试 |
| 验收报告 | QA 负责人 |
