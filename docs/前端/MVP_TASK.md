# 前端岗 - MVP 阶段工作明细

## 角色配置
- 游戏前端 (资深) × 2人
- 渲染/Shader (资深) × 1人

## 技术选型
- **渲染引擎**: Pixi.js (H5 兼容性最好)
- **状态管理**: 基于 ChangeDetection 机制
- **网络**: WebSocket + HTTP
