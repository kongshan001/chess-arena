## 核心玩法机制

### 1. 选秀阶段 (Carousel)
- 所有玩家从同一池子随机选择棋子
- 选秀顺序基于胜负场/生命值
- 可以获取特定棋子或装备

### 2. 棋子购买 (Shop Phase)
- 每回合固定获得金币 (基础5+连胜奖励)
- 免费刷新或花费金币刷新商店
- 购买棋子放置到备战区
- 备战区容量: 8-10个

### 3. 部署阶段 (Planning Phase)
- 将棋子从备战区拖到棋盘
- 棋子分前后排位置
- 某些棋子需要特定位置发挥最大效果

### 4. 战斗阶段 (Combat Phase)
- 自动战斗，没有玩家干预
- 棋子按AI行动：移动 → 寻找目标 → 攻击
- 技能自动释放

### 5. 羁绊系统 (Synergies)
- 同职业/同种族棋子触发增益
- 例如: 3战士 = +200护甲
- 多个同羁绊效果叠加

### 6. 升级系统
- 3个同星级棋子 = 1个更高星
- 3个1星 = 1个2星
- 3个2星 = 1个3星(传说)

### 7. 共享征召 (Shared Draft)
- 所有玩家vs同一敌人镜像
- 通过组合优化获取棋子
