# 策划岗 - MVP 阶段工作明细

## 角色配置
- 系统策划 (资深) × 1人
- 数值策划 (资深) × 1人

## 工作内容

### 系统策划

#### 第1周: 核心玩法设计
- [ ] 梳理自走棋核心乐趣点
- [ ] 设计棋子定位体系 (输出/控制/前排/辅助)
- [ ] 设计羁绊系统 (种族+职业)
- [ ] 撰写《核心玩法设计文档》
- [ ] 输出交互原型需求

#### 第2周: 战斗系统设计
- [ ] 设计战斗逻辑 (自动战斗+手动技能)
- [ ] 设计棋子升级/合成机制
- [ ] 设计装备系统 (MVP版)
- [ ] 撰写《战斗系统设计文档》

#### 第3周: 玩法调整
- [ ] 根据测试反馈调整数值
- [ ] 优化战斗体验
- [ ] 补充新手引导设计
- [ ] 更新平衡性参数

#### 第4周: 验收支持
- [ ] 配合 QA 完成功能验收
- [ ] 产出体验优化建议
- [ ] 规划后续版本内容

### 数值策划

#### 第1周: 数值框架
- [ ] 建立基础数值模型
- [ ] 设计棋子属性成长曲线
- [ ] 设计经济产出模型
- [ ] 产出《数值设计文档》

#### 第2周: 数值实现
- [ ] 配置棋子基础属性
- [ ] 配置羁绊数值加成
- [ ] 设计装备数值
- [ ] 搭建数值配置表

#### 第3周: 数值调优
- [ ] 根据测试数据进行数值平衡
- [ ] 调整经济产出节奏
- [ ] 优化棋子强度分布

#### 第4周: 数值固化
- [ ] 固化 MVP 版本数值
- [ ] 产出数值配置说明
