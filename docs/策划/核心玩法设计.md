# 核心玩法设计文档

## 1. 游戏概述

**Chess Arena (棋阵)** 是一款创新的 H5 自走棋游戏，融合了传统自走棋策略与快节奏竞技元素。

## 2. 核心玩法循环

```
选择棋子 → 布局棋盘 → 战斗自动进行 → 获得金币 → 升级/购买棋子 → 下一轮
```

## 3. 核心创新点

### 3.1 双模式战斗

- **经典模式**: 传统 8 人对战
- **快棋模式**: 4 人对决，节奏更快 (MVP阶段重点打磨)

### 3.2 技能系统创新

每个棋子拥有：
- **普通攻击**: 自动触发
- **主动技能**: 手动释放 (每个棋子 1-2 个)
- **终极技能**: 满足条件触发 (MVP 阶段 30% 棋子拥有)

### 3.3 站位策略

- 棋盘分前排/中排/后排
- 不同位置影响受击优先度
- 技能释放范围可影响站位选择

## 4. 棋子设计

### 4.1 棋子分类

| 定位 | 描述 | 占比 |
|------|------|------|
| 战士 | 高血量近战 | 25% |
| 法师 | 高伤害群攻 | 20% |
| 刺客 | 高机动切后 | 15% |
| 射手 | 远程持续输出 | 15% |
| 坦克 | 纯前排保护 | 10% |
| 辅助 | 治疗/增益 | 10% |
| 召唤师 | 召唤物作战 | 5% |

### 4.2 羁绊系统

- **种族**: 每个棋子 1-2 个种族
- **职业**: 每个棋子 1 个职业
- 凑齐 N 个同种族/职业触发羁绊效果

**MVP 版本羁绊设计**:
- 人类 (3/6): 攻速加成
- 精灵 (3/6): 闪避加成
- 兽族 (3/6): 攻击加成
- 亡灵 (3/6): 减甲效果
- 战士 (3/6/9): 护甲加成
- 法师 (3/6): 魔攻加成
- 刺客 (3/6): 暴击加成

## 5. 经济系统

### 5.1 金币来源

| 来源 | 数量 | 时机 |
|------|------|------|
| 基础收入 | 5/轮 | 每轮结束 |
| 胜利奖励 | 1-5 | 战斗胜利 |
| 连胜奖励 | 2-5 | 连胜时 |
| 利息 | 0-5 | 回合结束时 (25%利息) |

### 5.2 消费途径

- 购买棋子 (1/2/3/4/5金)
- 刷新棋子 (2金)
- 升级经验 (4/8/16金)
- 棋子加速 (1金)

## 6. 升级系统

### 6.1 玩家等级

- 1-9级，每级解锁：
  - 可上场棋子数
  - 刷新次数
  - 可购买棋子池

### 6.2 棋子升级

- 3 个同星级棋子 = 1 个下一星级棋子
- 3 星 + 3 星 = 2 星传说棋子 (MVP 不开放)

## 7. 战斗系统

### 7.1 战斗流程

```
准备阶段 (20秒) → 战斗阶段 → 结算阶段 → 准备阶段(下一轮)
```

### 7.2 战斗逻辑

- 自动普攻
- 能量满则释放技能
- 目标选择：最近敌方/血量最低/血量最高
- 死亡移除战场

### 7.3 胜负判定

- 一方棋子全部阵亡
- 存活棋子多者胜
- 若存活数量相同，剩余棋子总血量高者胜
