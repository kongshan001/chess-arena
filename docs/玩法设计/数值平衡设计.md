## 伤害计算

### 普攻伤害
```
实际伤害 = 攻击力 × (1 - 防御减伤) × 暴击修正
防御减伤 = 防御力 / (防御力 + 100)
暴击伤害 = 150%
暴击修正 = 1.5 (暴击时)
```

### 技能伤害
```
技能伤害 = 技能基础 × 技能倍率 × (1 - 魔抗)
```
