# Chess Arena 核心玩法设计

## 项目概述
Chess Arena（Chess Auto Battle）是一款8人回合制策略对战游戏，玩家通过摆放棋子组成阵容进行自动战斗。

## 核心机制

### 1. 8人对战模式
- 8名玩家两两对战，共4场对决
- 每场对决获胜者进入下一轮
- 半决赛获胜2人进入决赛
- 决赛产生冠军

### 2. 棋子系统
- 基础棋子：20种
- 每种棋子拥有独特技能
- 棋子分为：战士、法师、坦克、刺客、辅助5大类

### 3. 羁绊系统（7种基础羁绊）
| 羁绊 | 效果 | 棋子数 |
|------|------|--------|
| 战士 | 物理攻击+15% | 4 |
| 法师 | 法术伤害+15% | 4 |
| 坦克 | 防御+20% | 3 |
| 刺客 | 暴击率+10% | 3 |
| 辅助 | 治疗效果+20% | 2 |
| 守卫 | 保护相邻棋子 | 2 |
| 狂热 | 攻击速度+10% | 2 |

### 4. 战斗系统
- 自动回合制战斗
- 速度决定行动顺序
- 棋子根据AI自动释放技能
- 战斗时长：60-90秒

### 5. 数值配置
- 棋子等级：1-5星
- 经验值系统
- 羁绊激活条件
